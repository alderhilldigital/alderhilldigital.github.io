Building a Website

This course covers the essential skills required to understand how to create and maintain a website. HTML and CSS are the main languages used in developing web pages. This course uses these languages and other tools to create web pages with text, images, links, audio and much more. By the end of this course you will have fully function web site available online that you can add to and update when you want.

Application

There are no pre-requisites to applying for the course but a sufficient level of English and Maths is required to participate.
